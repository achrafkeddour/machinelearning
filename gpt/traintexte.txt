Introduction à Linux
Qu'est-ce que Linux ?
Linux est un système d'exploitation open source basé sur le noyau Linux, créé par Linus Torvalds en 1991. Contrairement aux systèmes d'exploitation propriétaires comme Windows ou macOS, Linux est librement accessible et personnalisable. Il est utilisé dans une large gamme d'appareils, des serveurs aux smartphones, en passant par les superordinateurs.

Naviguer dans le système de fichiers :

ls : Liste les fichiers dans le répertoire courant.
cd [chemin] : Change le répertoire courant.
pwd : Affiche le chemin complet du répertoire courant.
Gérer les fichiers et répertoires :

touch fichier.txt : Crée un fichier vide.
mkdir dossier : Crée un nouveau répertoire.
rm fichier.txt : Supprime un fichier.
cp source destination : Copie un fichier ou un répertoire.
mv source destination : Déplace ou renomme un fichier.
Gestion des utilisateurs et permissions :

whoami : Affiche le nom de l'utilisateur actuel.
sudo : Exécute une commande avec des privilèges élevés.
chmod [permissions] fichier : Modifie les permissions d'un fichier.
chown utilisateur:group fichier : Change le propriétaire d'un fichier.
Gestion des processus :

ps : Liste les processus en cours.
kill [PID] : Termine un processus.
top : Affiche une vue en temps réel des processus.
